Blinky v1.0 - Written by Dan (19th October, 2006)
_______________________________________________________________________________

Blinky is a level editor for the NES game, Ms. Pacman. It supports the editing
of all 32 levels, palettes, and a few data offsets that may prove useful in
hack development. Blinky also features a fairly simplistic automatic level
saving routine, which searches the ROM for free space to insert levels into
(due to the fact that the game uses compressed level data). Blinky also
automatically inserts an assembly hack written by DahrkDaiz
(http://dahrkdaiz.panicus.org) that enhances the games current palette system
and allows easy editing of the palette data for each level.
